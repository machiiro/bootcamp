ユーザー操作

ユーザー操作

sudo - 別のユーザーとしてコマンドを実行する

# root のみ許可されているコマンドを実行する
$ sudo service nginx restart

# 指定したユーザーで実行する
$ sudo -u user xxx.sh

su - 別のユーザーに切り替える

# root に切り替える、環境変数は元のユーザーのまま
$ su
# root に切り替える、環境変数は root のものになる
$ su -
# 指定したユーザーに切り替える
$ su - user
